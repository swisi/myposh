## Installation

### Windows

Install
``` cmd 
winget install JanDeDobbeleer.OhMyPosh
```

Update
``` cmd 
winget upgrade oh-my-posh
```

## Setup Terminal

### Windows

Get shell
```cmd
oh-my-posh get shell
```
Add Profile to powershell

``` notepad $PROFILE ```
Add ```oh-my-posh init pwsh | Invoke-Expression```
